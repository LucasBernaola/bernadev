# Lucas Bernaola Portfolio

Frontend portfolio built with Next.js, React and TypeScript. The site is focused on employability, with Health Care Plus as the featured product-style project and bilingual content for English and Spanish audiences.

## Getting Started

Install dependencies with pnpm:

```bash
pnpm install
```

Run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## Scripts

```bash
pnpm dev
pnpm build
pnpm lint
pnpm start
```

## Content

The bilingual copy lives in `app/content.ts`. The current homepage implementation lives in `app/page.tsx` and `app/page.module.css`.
